# Payment_Processing

This is basically a storefront. On the back end there are 
### Account:
<pre>
  {
    Name,
    AccountId,
    Email,
    Balance,
    DateOpened
  }
</pre>
### Product:
<pre>
  {
    Name,
    ProductId,
    ProductDescription,
    Price
  }
</pre>
### Item : Product (individual unit of a product)
<pre>
  {
    SKU,
    Attributes
  }
</pre>
### Attribute:
<pre>
  {
    Type: (color, size, etc.),
    Value
  }
</pre>
